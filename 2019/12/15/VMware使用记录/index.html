简介

虚拟机是通过软件模拟的具有完整硬件系统功能的、运行在一个完全隔离环境中的完整计算机系统。在虚拟机中，你可以安装各种操作系统、组建局域网等，模拟的完全是一个真实系统环境，不会对宿主机造成危害。

环境配置

  • win7
  • VMware 12 下载
  • ubuntu镜像 ubuntu-16.04-desktop-amd64.iso 下载

创建第一个虚拟机-Ubuntu

  • 在VMware中安装Ubuntu
  1. 选择 文件 -> 新建虚拟机 / Ctrl + N / 主页 -> 创建新的虚拟机
  2. 新建虚拟机向导：选择稍后安装操作系统，点击下一步
  3. 客户机操作系统：选择Linux 版本选择Ubuntu 64位
  4. 命名虚拟机名称：Ubuntu_20191216 ,位置：点击浏览选择对应存储位置，点击下一步。 （建议按照不同系统、不同功能分类设置虚拟机名称以及存储位置）
